Mission

Build a lean chart library for the graphs traders keep hunting for.

ChartVault is being built as a static site with a once-per-day update pipeline. The core tradeoff is deliberate: spend calls on reusable data once, then ship fast, shareable pages that stay cheap to operate.

Data model

Flat JSON files, aggressive reuse, no database, and no runtime backend.

Frontend

Astro 6, TradingView Lightweight Charts 5.1, plain client-side JavaScript, and static JSON datasets.

Update rhythm

One scheduled weekday refresh, lean call budgeting, and request logging on every run.
