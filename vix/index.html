How To Read It

  • Start with YTD or 1Y to frame the current move, then drop to 10D or 1M for the tactical read.
  • Use the 20, 50, and 200 day averages as context for trend strength, not as exact support or resistance lines.
  • Switch to % from view start when you want the move over the selected window instead of raw price.
